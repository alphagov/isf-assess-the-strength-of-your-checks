How you can tell if a source is authoritative

To be authoritative, the source must have data about either the identity or the evidence.

They must have received the data about the evidence from the organisation that issued it, either directly, or from another authoritative source. They’ll need to know enough about the identity or evidence to be able to confirm if the data you have is right. They’ll also need to make sure the data is up to date and hasn’t been tampered with.
